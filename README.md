## Project Overview
This repository contains shell scripts for:
1. **backup_script.sh**: Backs up a directory with timestamps.
2. **monitor_script.sh**: Logs CPU & Memory usage.
3. **download_script.sh**: Automates file downloads.

## How to Run
1. Open terminal and navigate to the folder.
2. Make scripts executable: `chmod +x script_name.sh`
3. Run the script: `./script_name.sh`
